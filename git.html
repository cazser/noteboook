1. git是什么？
    github的命令行工具
  2. 安装:可以安装 git for windows 可以安装cmder，cmder内置git
  3. git配置 命令行里运行:
    • git config --global user.name 你的英文名
    • git config --global user.email 你的邮箱
    • git config --global push.default simple
    • git config --global core.quotepath false
    • git config --global core.editor "code --wait"
    • git config --global core.autocrlf input
    保证在命令行输入code可以打开vscode 如果不可以建议重新安装 vscode 在安装时勾选添加到环境变量
  4. git能干什么版本控制。
    git init#新建仓库 #创建一个.git目录存放代码快照
    git add 文件名 #告诉git保存哪些文件
    .gitignore文件标记哪些文件不需要提交
    git status查看文件提交状况
    git commit -m <标签> #提交代码
